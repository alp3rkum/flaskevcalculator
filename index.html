EV Calculator

The Consumer-Friendly EV Calculator Program



Flask EV Calculator is a basic program to calculate various aspects of an electric vehicle. Flask EV Calculator specializes on light electric vehicles (LEV); like scooters, motorcycles and quadricycles.

Requirement Description
Python (ideally 3.10 and later)The programming language Flask EV Calculator was written in
Flask FrameworkA popular web routing framework for Python

Launch

You can run app.py or waitress_server.py (might need to change the port value if port 8080 occupied.)

To do so, open the waitress_server.py in any text editor and find the following line:

serve(app.app, host='127.0.0.1',port=8080)

You can change the port number to 8081, 8082... to make sure the application is served in production mode.

Usage

Upon launch, you can see that there are three different calculators (so far): range, acceleration and cost. You can choose any of these three calculators, enter the necessary inputs, and perform the calculations. You can look at the calculators' respective help pages to get more idea about how the calculations are done.
